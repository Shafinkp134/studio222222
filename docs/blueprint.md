# **App Name**: Gift Admin Pro

## Core Features:

- Product Management: Admin panel to add, edit, and remove gift products.
- Order Management: View customer orders and update checkout details with transaction IDs.
- Secure Login: Secure login for administrators via Google Authentication.
- Admin Panel: Dashboard for managing products, orders, and user accounts.
- Database: Firebase database to store product, order, and user data.

## Style Guidelines:

- Primary color: Soft peach (#FFDAB9) to evoke warmth and friendliness.
- Background color: Very light peach (#F8E8DE) for a subtle, inviting base.
- Accent color: Light gold (#E6BE8A) for highlights and key CTAs, giving a touch of luxury.
- Body and headline font: 'Alegreya' for a balanced, elegant look.
- Use elegant, hand-drawn style icons to match the friendly aesthetic.
- Clean and intuitive layout for easy navigation.
- Subtle transitions and animations to enhance user experience.
